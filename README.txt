
Leen Coffee - Ultra Professional Website
========================================

Files ready to deploy on Netlify. Just upload the folder contents (or the zip) to Netlify Drop.

Structure:
- index.html  (landing)
- menu.html   (menu with modal viewer)
- assets/css/style.css
- assets/js/app.js
- assets/img/logo.png  (your uploaded logo)
- assets/img/hero.jpg  (hero image)
- data/config.json
- data/menu.json

To customize: edit data/config.json and data/menu.json
